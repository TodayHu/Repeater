# Repeater

Swift timer on GCD

## Installation

Repeater is now available only through pod file with `use_frameworks!` option . To install
it, simply add the following line to your Podfile:

```ruby
pod 'Repeater', :git => 'https://github.com/Koshub/Repeater.git'
```
